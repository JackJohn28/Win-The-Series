# 🏆 Win The Day: The Series

A gamified habit-tracking PWA (Progressive Web App) that transforms weekly goals into a professional sports playoff experience. 

## 🏟️ The Concept
Inspired by the "Win The Day" philosophy, this app moves away from traditional streaks. Instead, users compete in a **7-Game Series** every week.
* **Goal:** Secure 4 wins before 4 losses to "Clinch the Week."
* **Venue Logic:** Experience "Home" vs. "Away" games with dynamic UI themes that reflect the difficulty of the week (e.g., the Monday-Wednesday mid-week "road trip").
* **Championship Mode:** Achieving a 4-0 sweep unlocks a 3-day Championship Series for ultimate bragging rights.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Dark Theme), JavaScript (ES6+)
* **Backend/Database:** Firebase Firestore
* **Authentication:** Firebase Auth
* **Hosting:** Firebase Hosting / GitHub Pages (PWA)

## 🎮 Key Features
- **11:00 PM Check-in:** A nightly binary choice: Did you Win or Lose?
- **Stat-Driven Recaps:** Standardized dropdown menus to track "Key Contributors" (Wins) or "Turnovers" (Losses) for trend analysis.
- **Home/Away Dynamics:** - **Home (Sat, Sun, Thu, Fri):** Favorable UI accents to keep momentum.
  - **Away (Mon, Tue, Wed):** Aggressive UI accents to simulate the "Road Grind."
- **24-Hour Grace Period:** One-day "overtime" to log results before an automatic forfeit is recorded.
- **Scouting Report:** Data visualization showing your series record, win percentage, and "MVP" habits.
